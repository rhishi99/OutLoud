📈 Build Journey 🎛️ Live Visualizer + Config ⭐ GitHub 📖 Full README

Quick install (Claude Code)

/plugin marketplace add rhishi99/OutLoud
/plugin install speaker@outloud

Use /speak, global hotkey, or enable /speak on for auto-narration. Edge-tts (natural) by default with fast native fallback.

This site is served via GitHub Pages. To enable (or re-enable):
Repo → Settings → Pages → Source: Deploy from a branch → main / root → Save.
